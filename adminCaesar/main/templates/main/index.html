Шифр Цезаря

Заменяет каждую букву в тексте на другую, смещённую на фиксированное число позиций в алфавите.

Шифр AES

Симметричный блочный шифр с длиной ключа 128, 192 или 256 бит.

Шифр RSA

Асимметричный алгоритм, использующий простые числа для генерации

Шифр BLOWFISH

Симметричный блочный шифр с переменной длиной ключа.

Шифр VIGINERE

Расширение шифра Цезаря, использующее ключевое слово для изменения смещения каждой буквы.

Шифр MD5

MD5 — хэш-функция, преобразующая данные в 128-битный хэш

Шифр SHA-256

Криптографическая хэш-функция для проверки целостности данных.

Шифр DES

Старый стандарт симметричного шифрования с 56-битным ключом.

Шифр TRIPLE-DES

Расширение DES с тройным шифрованием.

Шифр ONE TIME PAD

Идеальный шифр, использующий случайный одноразовый ключ.
